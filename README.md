# Schedula | Modern Healthcare Dashboard

Schedula is a premium, user-centric healthcare management platform designed to streamline the patient experience. With a focus on accessibility and modern aesthetics, Schedula allows users to find specialists, book appointments, and manage medical records with ease.

## ✨ Features

- **Doctor Discovery**: Search and filter through a curated list of specialists in various fields (Cardiology, Psychology, Pediatrics, etc.).
- **Smart Booking**: Intuitive date and time selection with a floating confirmation bar for absolute clarity.
- **Detailed Profiles**: Access professional biographies, ratings, and experience details for every healthcare provider.
- **Personalized Dashboard**: A comprehensive view of upcoming appointments, medical records, and vitals.
- **Modern UI**: A sleek, dark/light hybrid interface centered around a premium purple aesthetic.
- **Secure Access**: Simple login and signup flows with a dedicated logout system.

## 🚀 Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript, CSS3
- **Icons**: [Lucide Icons](https://lucide.dev/)
- **Typography**: [Outfit (Google Fonts)](https://fonts.google.com/specimen/Outfit)
- **State Management**: LocalStorage for persistence
